### React에서 `Emnet` 사용하기

`JSX` 작성할때 `Enmet` 안되서 불편했음.

1. VSCode 설정 열기

   - Windows: `Ctrl + ,`
   - Mac OS: `Command + ,`

2. WorkSpace 설정에 들어가기

3. settings.json 열기

4. 아래 코드 넣어주기

```
{
   "emmet.includeLanguages": {
      "javascript": "javascriptreact"
   }
}
```
